# taskmaster
Este é o taskmaster, meu primeiro projeto criado por iniciativa 100% própria com o intuito de servir como aprendizado. A proposta do taskmaster é gerenciador de tarefas extremamente simples estilo com identidade visual fortemente inspirada Trello, ferramenta famosa. A ideia do projeto é ter uma interface e funcionalidade o mais básicas possível e evoluir gradualmente.

## Detalhes técnicos do projeto no estado atual
\begin{itemize}
    \item Front-end feito em Angular 21
\end{itemize}

## Recursos presentes atualmente
\begin{itemize}
    \item Front-end pronto e funcional sem persistência dos dados
    \item Interface limpa e minimalista
    \item Funções básicas de criar, editar e excluir tarefas
    \item Trocar o estado das tarefas
    \item Possibilidade de mover as tarefas usando drag and drop (tanto para reordenar quanto para trocar de estado)
\end{itemize}

## Recursos pretendidos para o futuro
\begin{itemize}
    \item Persistência dos dados em um banco PostgreSQL
    \item Back-end feito em Java (Spring Boot)
    \item Sistema de autenticação com OAuth 2
\end{itemize}

## Instruções para rodar o projeto
Para executar o projeto, clone o repositório e abra um terminal dentro da pasta "front". Depois disso, instale as dependências através do comando \textbf{npm install}. Após a devida instalação das dependências, execute o comando \textbf{ng serve}. Quando a mensagem \textit{"Local:   http://localhost:4200/"} aparecer escrita no seu terminal, a aplicação já estará disponível neste endereço.
