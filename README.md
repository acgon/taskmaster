# taskmaster
Este é o taskmaster, meu primeiro projeto criado por iniciativa 100% própria com o intuito de servir como aprendizado. A proposta do taskmaster é gerenciador de tarefas extremamente simples estilo com identidade visual fortemente inspirada Trello, ferramenta famosa. A ideia do projeto é ter uma interface e funcionalidade o mais básicas possível e evoluir gradualmente.

## Detalhes técnicos do projeto no estado atual
- Front-end feito em Angular 21

## Recursos presentes atualmente
- Front-end pronto e funcional sem persistência dos dados
- Interface limpa e minimalista
- Funções básicas de criar, editar e excluir tarefas
- Trocar o estado das tarefas
- Possibilidade de mover as tarefas usando drag and drop (tanto para reordenar quanto para trocar de estado)

## Recursos em fase de desenvolvimento
- Back-end feito em C#

## Recursos pretendidos para o futuro
- Persistência dos dados em um banco PostgreSQL
- Sistema de autenticação com OAuth 2

## Instruções para rodar o projeto
Para executar o projeto, clone o repositório e abra um terminal dentro da pasta "front". Depois disso, instale as dependências através do comando __npm install__. Após a devida instalação das dependências, execute o comando __ng serve__. Quando a mensagem _Local:   http://localhost:4200/_ aparecer escrita no seu terminal, a aplicação já estará disponível neste endereço.
